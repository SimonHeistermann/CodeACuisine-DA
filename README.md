# Code à Cuisine

> AI-powered recipe generator that turns your ingredients into three instant recipes — training project from Developer Akademie.

## Disclaimer

This is a **training and portfolio project** built as part of my education at [Developer Akademie](https://developerakademie.com/). It is not a commercial product and is not intended for real-world use. No real orders, services, or transactions are processed. The AI-generated recipes are for demonstration purposes only.

## About

Code à Cuisine is an interactive web app that transforms your available ingredients into three instantly cookable recipes — generated by an AI-powered workflow using Google Gemini. All recipes are stored in Firebase Firestore and can be browsed in a cookbook with cuisine filtering and a favorites system.

## Tech Stack

- **Frontend:** Angular 21 (standalone components, new control flow)
- **Language:** TypeScript (strict mode)
- **Styling:** SCSS + Tailwind CSS 4
- **Database:** Firebase Firestore
- **AI Backend:** n8n workflow on Render.com (Google Gemini)
- **Rate Limiting:** Redis (via n8n)

## Features

- **AI Recipe Generation** — Enter ingredients and preferences, receive 3 validated recipes
- **Cookbook** — Browse all saved recipes, filtered by cuisine, sorted by popularity
- **Favorites** — Hybrid storage: localStorage for personal likes, Firestore for global counters
- **Recipe Detail** — Full ingredient lists, step-by-step directions, nutritional info, chef role assignments
- **Deduplication** — Deterministic recipe signatures prevent duplicate entries
- **Responsive Design** — Mobile-first with fluid typography and spacing

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

```bash
npm install
```

### Configuration

Set up environment files in `src/environments/`:

- `environment.ts` / `environment.development.ts` — webhook URL
- `firebase.config.ts` — Firebase project credentials

### Running

```bash
npm start
```

Available at [http://localhost:4200](http://localhost:4200).

### Building

```bash
npm run build
```

Output: `dist/code-a-cuisine/`

### Testing

```bash
npm test
```
